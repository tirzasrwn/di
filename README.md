# di

## Description

Understanding dependency injection using nest js.

## Generate

This code was generated using this step.

```bash
$ npm install -g @nestjs/cli
$ nest new di
$ nest g module computer
$ nest g module cpu
$ nest g module disk
$ nest g module power
$ nest g service computer
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
